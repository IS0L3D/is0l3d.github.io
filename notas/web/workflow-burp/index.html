Workflow rapido con Burp

Flujo minimo para mapear y priorizar pruebas web.

2025-12-07 · Web, Pentesting

Alcance

Definir host, rutas criticas y cuentas de prueba antes de interceptar trafico.

Proxy y target

  1. Configurar navegador con proxy de Burp.
  2. Confirmar certificado CA instalado.
  3. Construir arbol en Target.

Repeater

Enviar solicitudes clave para validar:

  • Controles de autorizacion.
  • Validaciones de entrada.
  • Respuestas inconsistentes.

Priorizacion

Empezar por endpoints con:

  • cambios de estado.
  • subida de archivos.
  • acceso a datos sensibles.

Registro

Guardar request/response y evidencia por cada hallazgo.
